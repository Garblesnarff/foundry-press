## Description

100% local. 100% offline. 100% yours.

Every document is raw material. Foundry Press forges it into publication-ready output — a native macOS Typst editor with live PDF preview, bringing the modern document composition language to the Mac with the interface it deserves. No browser editor. No Electron. No LaTeX compiler sprawl. Just a lean, fast, native SwiftUI app that compiles Typst in milliseconds.

Typst is the modern LaTeX: structured markup, powerful scripting, beautiful output, and a compiler that runs in a fraction of the time. But Typst deserved a better editor than a browser tab. Foundry Press is that editor — split-pane live preview, syntax highlighting, template library, iCloud sync, and PDF export that opens in your workflow instantly.

**Write in Typst. See it instantly. Export to PDF.**
- Split-pane editor: write on the left, watch the PDF render on the right in real time
- Full syntax highlighting for Typst markup, code blocks, and math expressions
- Template library: start from academic paper, résumé, invoice, report, or blank document
- iCloud sync: your documents on every Mac, always up to date
- Export to PDF with one keystroke — resolution-perfect, print-ready output
- Custom fonts: install and use any font in your Typst documents
- Error panel: compilation errors highlighted inline with line-level markers

**One compiler. One forge.**
The Typst Rust compiler is bundled directly in the app — no package manager, no system dependencies, no `brew install typst`. The forge is self-contained. Documents compile in under 100ms. The live preview re-renders as you type.

**Built for academics, technical writers, and anyone who outgrew LaTeX.**
The PhD student writing their thesis in Typst instead of overleaf. The engineer documenting an API spec. The freelance writer producing client reports with structured formatting. Foundry Press is the forge for professionals who want LaTeX-quality output without LaTeX's ceremony.

Your documents never leave your machine unless you export them. No telemetry. No phone-home. No surprises.

The markup goes in. The document comes out.
